# Code Lab Cinema

The same concept as Code Lab TV, **Code Lab Cinema** lets you learn programming through mini-games.  
Instead of focusing on solo gameplay, it emphasizes **offline multiplayer battles**, turning coding into a competitive and social experience.

Give url with the code of your friend or player you trust in and let's them fight.   
Note that GDScript have all permissions be careful.   
See information about what is the reach of GDScript on PC and Android.   

The vision is simple:   
* **Train** your coding skills in **Code Lab TV**.   
* **Have fun** observing and playing in **XR**.   
* **Compete** in **Code Lab Cinema** events and **Twitch TV tournaments**.   

A standalone **Code Editor** lets you write, test, and load your code without opening the Godot Editor.   
Repository: https://github.com/EloiStree/APInt128x64
